📌 Fastify CRUD API

API leve e eficiente desenvolvida com Node.js e Fastify, focada na realização de operações CRUD (Create, Read, Update, Delete) para aplicações web e móveis.

O projeto foi criado com o objetivo de oferecer um ambiente minimalista, rápido e escalável, ideal para sistemas que exigem alto desempenho e respostas ágeis.

🚀 Funcionalidades


📥 Criação de registros (Create)
📄 Listagem de dados (Read)
✏️ Atualização de informações (Update)
❌ Remoção de dados (Delete)
⚡ Alta performance com Fastify
🛠️ Tecnologias
Node.js
Fastify


🎯 Objetivo

Demonstrar a construção de uma API simples e performática, servindo como base para projetos maiores e aplicações modernas.

📈 Possíveis melhorias
Integração com banco de dados
Autenticação e autorização
Validação de dados
Deploy em nuvem
